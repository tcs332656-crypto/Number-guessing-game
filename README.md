# Number guessing game

## Description :

This is a simple python game where the computer picks a random number from 1 to 100 and then the user guesses the number , every time the user guesses wrong ,the number of tries increase , the game continues till the user guesses the number right .Throughout the game the user gets hints from the computer to guess higher or lower according to the number decided by the computer.

## Language used 
-Python

## Features
-Random module
-loops(while and if)
-guess counter
-higher and lower hints
